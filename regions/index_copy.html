How does it work?

CSS Regions allows you to take content from a source element and then format it into a series of regions. You can also get really fancy and make text wrap into regions of specific shapes (rectangle, circles, ellipses, or polygons).

** It should also be noted that CSS Regions is not widely supported and is mostly disabled by default in most browsers. Seems to work fine in Chrome 18.

To implement CSS Regions:
  • In CSS, define an element with the following:
    -webkit-flow-into: main-thread

    Example:
    #source{
    -webkit-flow-into: main-thread;
    text-align:justify;
    }
  • Then define an element class, or "region" referencing the source element:
    -webkit-flow-from: main-thread;

    Example:
    .region{
    -webkit-flow-from: main-thread;
    }
  • Create source and region elements.
